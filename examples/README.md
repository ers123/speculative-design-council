## 4개 예시의 공통 구조

모든 예시는 동일한 6-section 구조:
1. **세계 환경** — 시기·장소·핵심 사건 (2-3 문장)
2. **에이전트 명단** — 5명, 다양한 소속 (정부·민간·시민단체·감독기관·지하 운동)
3. **협업 프로토콜** — 우선순위·충돌 해결·검열·기록
4. **시민 인터페이스** — 누가 어떤 권한·거부권 가지는가
5. **책임 소재** — 잘못된 결정의 책임 주체
6. **변경 이력**

## 4개 예시의 공통 비평 패턴

각 예시 끝에 **비판 포인트** 명시:
- HAPPY GATE: 시민 거부권 부재
- COCOO: 자녀 동의 없는 데이터 수집
- SPLIT: 알고리즘 공개 차단으로 차별 추적 불가
- SelectON: 필터링되는 사람의 가시성 상실

→ 좋은 AGENTS.md는 *"이상적 거버넌스"* 를 묘사하는 게 아니라 **불완전함·갈등·균열**을 드러냄. spec design 정신.
